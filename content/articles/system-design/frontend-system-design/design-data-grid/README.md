# Draft: Design a Data Grid

Building high-performance data grids for large datasets.

## TLDR

- Virtualization enables rendering millions of rows
- Memoization prevents unnecessary re-renders
- Accessibility requires keyboard navigation

## Outline

1. Large dataset rendering: virtualization, windowing
2. Column features: sorting, filtering, resizing, reordering
3. Row features: selection, expansion, grouping
4. Cell features: editing, custom renderers
5. Fixed columns/rows: pinning, freezing
6. Performance: memoization, selective rendering
7. Accessibility: keyboard navigation, ARIA
8. Export: CSV, Excel, PDF generation
