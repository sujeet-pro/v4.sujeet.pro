# Draft: Design Spotify Music Streaming

Building an audio streaming platform like Spotify.

## TLDR

- Audio streaming requires bandwidth-adaptive delivery
- Offline mode needs sync and license management
- Discovery drives user engagement

## Outline

1. Audio streaming: formats, quality levels, bandwidth adaptation
2. Content delivery: CDN, P2P (historical), caching
3. Playlists: collaborative playlists, ordering
4. Recommendations: Discover Weekly, algorithmic playlists
5. Offline mode: download, license management, sync
6. Search: tracks, artists, albums, playlists
7. Social features: friend activity, shared playlists
