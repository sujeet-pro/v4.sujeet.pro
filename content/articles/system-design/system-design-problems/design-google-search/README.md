# Draft: Design Google Search

Building a web search engine at scale.

## TLDR

- Crawling at scale requires distributed coordination
- Indexing enables fast query processing
- Ranking determines result quality

## Outline

1. Web crawling: scale, freshness, completeness
2. Indexing: inverted index, document processing
3. Ranking: PageRank, relevance signals, ML ranking
4. Query processing: parsing, spelling correction, synonyms
5. Serving: index sharding, caching, real-time features
6. Personalization: search history, location
7. Specialized search: images, news, shopping
