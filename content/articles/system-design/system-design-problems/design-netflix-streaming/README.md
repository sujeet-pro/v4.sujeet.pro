# Draft: Design Netflix Streaming

Building a video streaming platform at Netflix scale.

## TLDR

- Open Connect CDN brings content close to users
- Adaptive bitrate ensures smooth playback
- Personalization drives engagement

## Outline

1. Content delivery: Open Connect CDN, edge servers
2. Adaptive bitrate: quality selection algorithms
3. Encoding pipeline: per-title encoding, shot-based encoding
4. Personalization: recommendation system, artwork personalization
5. Device support: encoding profiles, DRM
6. Offline viewing: download management, license handling
7. Playback resilience: buffering, quality switching
