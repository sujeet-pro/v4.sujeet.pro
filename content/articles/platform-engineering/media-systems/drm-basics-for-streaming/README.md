# Draft: DRM Basics for Streaming Media

Foundations of DRM, license servers, and playback protection.

## TLDR

- DRM encrypts content and enforces playback policies
- License servers and key rotation are central
- Multi-DRM support is required for broad device coverage

## Outline

1. DRM concepts and threat model
2. Packaging and encryption workflows
3. License server architecture
4. Multi-DRM support and compatibility
5. Policy enforcement and revocation
6. Operational considerations
